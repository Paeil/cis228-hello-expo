Ham-al (Your Personal Sous Chef)

Description
Ham-al is a mobile application designed for the modern kitchen. It is a personalized meal planner and recipe cookbook that features iconic Filipino Gourmet dishes like Chicken Adobo, Sinigang, and Lechon Kawali. The app helps users organize their weekly meals, manage grocery lists, and save their favorite recipes in a digital cookbook.

How to Install and Run
Follow these simple steps to get the app running on your device:

1. Install Dependencies: Open your terminal in the project folder and run:
npm install

2. Start the Project: Run the following command to start the Expo server:
npx expo start

3. View the App: Scan the QR code with your Expo Go app (Android) or Camera (iOS) to see Ham-al in action!
